# jd-scfa
This repository contains raw proteomics data and R scripts for data processing and analysis used in the study titled: "**Exercise training induces mitochondrial biogenesis, while high fat diet increases the ability of mitochondria to use long and short-chain fatty acids**".

## Files
- 'jd-scfa_raw data.xlsx' - Raw proteomics data
- 'jd-scfa_2-ANOVA.R' - Script for statistical analysis (2-way ANOVA)
- 'jd-scfa_PCA+biplot.R' - Script for PCA and biplot
- 'jd-scfa_volc plot + DEP.R' - Script for volcano plot and differential protein expression between individual groups

Experimental procedures and study details are fully described in the final published version of the manuscript. If you have any questions, please contact: ghollowa@uoguelph.ca
